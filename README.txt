테라 클랙식 소탕 매크로!!

기본 사냥터는 숲 매머드고, 원하는 사냥터로 변경 가능.


1. 지원단말 : 갤럭시S8, 갤럭시노트8
2. 준비사항 : 개발자 모드 > 전원케이블 연결시 화면 안꺼지는 설정
3. 테라 앱 아이콘은 바탕화면 왼쪽 아래에
4. 기본 사냥 필드 : 숲 매머드 (케릭을 매머드 사냥터로 옮겨야함)
5. 신규 기능
  - 앱 실행 시간 지정
  - 자동 로그인 (앱 업데이트, 공지사항, 로그인 후 첫 화면에 과금 광고 등등 다 자동 넘어감)
  - 자동 오토 클릭
  - 재접속시 다시 오토 시작
  - 소탕 완료시 자동 수령
  - 다른 기기 중복 접속 시 재 로그인
